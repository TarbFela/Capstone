# A Temperature Programming Power & Control System for Desorption Energy Measurement Experiments

This is my electrical & computer engineering capstone as well as my honors thesis at the University of Maine. The hardware includes a number of power cards driven by PWM signals from a control board. Another board provides a serial interface to a Balzers mass spectrometer QMS 112 interface. The software for this project is largely written for the RP2350 microcontroller (like that found on the Raspberry Pi Pico 2) on a Pimoroni PGA2350 devboard/daughtercard. The PGA2350 plugs into the controller motherboard called ADPC for "Analog and Digital Processing and Control". This board incorporates, in addition to other instrumentation, MCP3462R and MCP3562R ADCs and an ADA4255 PGIA; all three of these provide SPI interfaces. I'm writing drivers for these devices which can be found in `src2/`. Functionalities are loosely (and currently rather haphazardly, with deprecation, poor compartmentalization, etc) broken out into separate headers/source files. 

I'll probably post hardware files somewhere at some point. The READMEs are currently acting as in-progress note-taking and as in-progress documentation. This project is meant to be used after I graduate by researchers in the Frederick Lab at FIRST. The code is roughly written with extensibility in mind.
